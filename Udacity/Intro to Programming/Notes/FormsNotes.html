Forms

Forms are text boxes, radio boxes, check boxes, etc. Form tags look like this: 

			<form>
			</form>

<form action=" ">

Form has takes its own attribute, Action, that controls where you want the form to submit to. The Following is a form tag with the action attribute set submit to /foo: 
			<form action="/foo">
				<input name ="q">
				<input type ="Submit">
			</form>
<form action="http://google.com/search"> will send whatever is entered in the form to google's search page and return the results.

<input type="password">

"Text" is the default type parameter in a form's input tag. Before, the input was text, so this was fine, however, if you want to take a password, the form input would look like: 
			<form>
				<input type="password" name ="q">
				<input type ="Submit">
			</form>
And the input into the form is hidden as you type it.
The input is still in the URL however, so this doesn't mean the password is secure.

<input type="radio">

Another input type is a radio button. Radio buttons are coded like: 
			<form> 
				<input type="radio" name ="q">
				<input type="radio" name ="r">
				<input type="radio" name ="s">
				< br > 
				<input type ="Submit"> 
			</form>
And display like:

Radio buttons act like checkboxes unless you give the Radio buttons the same name. When they have the same name, you are asking for one answer, and not allowing more than one. Look at the difference in the radio buttons below vs the radio buttons above.

The first set of radio buttons had the q, r, and s names. The 2nd set all have the name q. However, we need a way to distinguish between which choice was selected. To do this, we add a value attribute like the following: 
			<form> 
				<input type="radio" name ="q" value="one">
				<input type="radio" name ="r" value="two"> 
				<input type="radio" name ="s" value="three">
				< br > 
				<input type ="Submit"> 
			</form>
And the display looks the same. However, look at the difference in the URL changes when clicking the radio buttons below:

Notice how ?q=one, ?q=two, or ?q=three is added to the URL. But, the user needs to see what they are chosing. Enter the label elements.

Dropdown boxes

Dropdown boxes use a <select> tag. They are formated like the following: 
			<form>
				<select name="q">
					<option value="1"> the number one </option>
					<option value="2"> the number two </option>
					<option value="3"> the number three </option>
				</select>
				<br>
				<input type="submit">
			</form>
